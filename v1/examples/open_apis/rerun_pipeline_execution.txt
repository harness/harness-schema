API to rerun a pipeline execution

path - execute/rerun/{execution-id}

Curl -
curl --location --request POST 'http://localhost:12001/api/v1/orgs/default/projects/fdsf/pipelines/pipetest14/execute/rerun/2sulijX2ROiaXnPDxW_nKg?module=cd&repoIdentifier=testt12&branch=main&connector_ref=account.rer&repo_name=testt12' \
--header 'Accept: */*' \
--header 'Accept-Language: en-GB,en-US;q=0.9,en;q=0.8' \
--header 'x-api-key: token' \
--header 'Harness-Account: kmpySmUISimoRrJL6NL73w' \
--header 'har: https://localhost:8181/ng/account/kmpySmUISimoRrJL6NL73w/cd/orgs/default/projects/njh/setup/resources/template-studio/StepGroup/template/-1/' \
--header 'Content-Type: application/json'
